# E-Commerce-Power-BI_Dashboard
This Power BI dashboard provides a comprehensive analysis of e-commerce sales performance across various regions, product categories, and payment methods. The goal is to track key performance indicators (KPIs) like revenue, average customer ratings, discount distributions, and delivery efficiency to drive data-backed business decisions.
